<SectionTitle>Creating a Theme</SectionTitle>

If you wish to create your own theme use the provided `createTheme` function. Each component provides its own theme
 that can be overridden. Please see the respective component documentation for a full list of the available theme
  overrides.

```
import { createTheme } from '@rtkwlf/ui';

const myTheme = createTheme({... overrides});
```

<SectionTitle>Using a Theme</SectionTitle>

Once you have created a new theme you can provide it to the components via the `ThemeProvider` API found in `styled
-components`.

```
import { ThemeProvider } from 'styled-components';

const App = () => (
  <ThemeProvider value={myTheme}>
    <Button>Themed Button</Button>
  </ThemeProvider
)
```
