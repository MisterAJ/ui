<Title
  title="getScrollPosition"
  subtitle="Get the scroll position for a provided element"
/>

There are many experiences that require the need for a component or function to know the current scroll position of
 an element relative to its position in the DOM. This function aims to make it easy to get the current scroll
  position for things like `window.scrollTo()`.

<SectionTitle>Example</SectionTitle>

```
import { getScrollPosition } from '@rtkwlf/ui';

const elementScrollPosition = getScrollPosition(element);

window.scrollTo({...elementScrollPosition, behavior: 'smooth'});
```
